## 핵심
- Union-Find, Disjoint set
- 백트래킹 완전 탐색도 가능
- 노드 시작 번호 주의해야함
## 사고흐름
1. 노드-노드 간의 이동 시에 경로는 상관없음 -> 연결 여부만 중요
2. 노드가 '연결'만 되면 이동 가능 -> 그래프가 연결되어있다 == 같은 집합에 속함
3. 같은 집합인지 판단 -> 유니온 파인드 사용해서 모든 노드 돌아서 부모 정의
